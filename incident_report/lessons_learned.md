# Lessons Learned

- Phishing remains the primary initial access vector; targeted awareness and simulated phishing tests should be prioritized.
- Telemetry gaps: insufficient EDR coverage on contractor-managed endpoints delayed detection.
- Incident response playbooks should include staged forensic triage for evidence preservation.

Action items:
- Expand EDR deployment to all endpoints and enforce central logging.
- Implement automated hunting for C2 TLS fingerprints and rare User-Agent strings.
